
indexlwl 
===================================  
  不知道取个啥名字，随便来
  
    
描述 
-----------------------------------  
  可以将当前文件夹下的子文件的内容生成一个Json对象<br/>
  且实时监听文件内容的变化，生成的对象在文件夹上一层的code文件里
    
### 使用方法  
		npm install indexlwl 
		lwl D:flie    ---- 就是对D盘下的flie文件夹下面的文件遍历
